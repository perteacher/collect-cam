# 채집 카메라 (Collect Cam)

후면 카메라로 사물을 겨누고 **채집** 버튼을 누르면, 그 순간의 화면을 찍어 배경을 지운 뒤(누끼) 투명 PNG 컷아웃이 아래 트레이로 톡 떨어져 쌓이는 채집 게임입니다. 인스타 릴스 "채집 앱 / 킹받는 프로그램" 감성을 단일 HTML 파일로 구현했습니다.

- 빌드 도구 / 외부 라이브러리 **없음** (순수 HTML/CSS/JS, `index.html` 하나).
- 배경 제거는 **모서리 플러드필** 방식(머신러닝 모델 다운로드 없음) — 오프라인에서 즉시 동작. 단색·균일한 바닥/책상 배경에서 가장 잘 됩니다.
- 카메라는 **https(보안 연결)** 에서만 작동합니다. GitHub Pages는 https라 그대로 됩니다. (로컬은 `localhost`도 OK.)
- 튜닝 값은 `index.html` 상단 상수에서: `MAX_DIM`(처리 해상도), `TOLERANCE`(배경 유사도 임계값).

## GitHub Pages 배포

저장소에 `index.html`을 올린 뒤 **Settings → Pages → Branch: `main` / 폴더 `/ (root)` 선택 → Save** 하면 `https://<사용자명>.github.io/<저장소명>/` 주소로 열립니다. (휴대폰에서 그 https 주소로 접속해 카메라 권한을 허용하세요.)
